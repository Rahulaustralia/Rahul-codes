CSS Transition

For Smooth Transformation

Transition Properties

  1. Transition-property: hight,width,color,background-color, all
  2. Transition-duration: 1s,2s,0s,0.1s/5ms
  3. Transition-timing-function:linear/ease/ease-in/ease-out/ease-in-out
  4. Transition-delay: 0s,2s,5ms

Short hand Properties

Transition: Transition-property,Transition-duration,Transition-timing-function,Transition-delay

Example:- Transition: all 2s linear 0.2s;



CSS- Transform

*transform means changing one state to another state of an element

CSS-Animation

  • The continuous transformation is known as animation
  • In CSS for defining animation of an element we have to follow the keyframe rules by using a speeid selector
  • @keyframes
  • Inside keyframe we are defining the styles of an elements animation by Percentage(%) values


    • Animation Properties

    1. animation-name
    2. animation-duration: 1s, 2ms
    3. animation-timeing-function : linear/ ease/ease-in/ease-out/ease-in-out
    4. animation-delay: 0s/ 1ms
    5. animation-iteration-count: 1/2/3
    6. animation-direction
